# Weather-App
A weather app to display current weather of a city. 

It uses an API from openweathermap.org and requests for the weather of a city by it's ID. The response is JSON which contains the data of the weather. Data from the response is parsed and displayed. 
UI is made using python Tkinter library.

Requirements - 
1. Python 2.7 

How to run - 
1. Get your api key from http://openweathermap.org/ and paste it in the user_api
2. Run weather_app.py
